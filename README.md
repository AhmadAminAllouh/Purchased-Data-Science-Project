# Purchased-Data-Science-Project
For this project, I utilized data from Kaggle and performed data science techniques to understand and organize the dataset. I then applied machine learning models—such as logistic regression, k-NN, and SVM—to predict future values, while adhering to a logical modeling workflow and ensuring that no data leakage occurred.

You can see the original data I worked on here: https://www.kaggle.com/datasets/dragonheir/logistic-regression
